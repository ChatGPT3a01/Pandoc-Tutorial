# Pandoc 文件轉換大師：從零開始的實作教學

> 掌握「文件界的瑞士刀」，輕鬆實現 Markdown、Word、PDF、HTML 格式互轉

## 課程簡介

Pandoc 是一款功能強大的「文件萬用轉換器」，能在不同格式之間轉換文件。本課程將帶你從零開始，完整掌握 Pandoc 的各種技巧。

**適用環境：** Windows
**語言：** 繁體中文

## 課程目錄

| Part | 標題 | 說明 |
|:----:|------|------|
| 0 | [Markdown 基礎教學](Pandoc教學_Part0_Markdown基礎.html) | 從零學習 Markdown 語法，為 Pandoc 打好基礎 |
| 1 | [Pandoc 入門與安裝](Pandoc教學_Part1_入門與安裝.html) | 認識 Pandoc 的強大功能，完成 Windows 環境安裝 |
| 2 | [基礎格式轉換實戰](Pandoc教學_Part2_基礎格式轉換.html) | Markdown ↔ Word ↔ HTML 互轉技巧 |
| 3 | [PDF 輸出與中文字型](Pandoc教學_Part3_PDF與中文字型.html) | LaTeX 安裝、XeLaTeX 引擎、繁體中文完美顯示 |
| 4 | [進階功能：公式與圖表](Pandoc教學_Part4_進階功能.html) | LaTeX 數學公式、圖片插入、Mermaid.js 整合 |
| 5 | [自動化批次腳本](Pandoc教學_Part5_自動化腳本.html) | 批次檔與 PowerShell 腳本，一鍵完成多格式轉換 |

## 學習路徑建議

- **完全新手：** 建議從 Part 0 開始學習 Markdown 語法，再進入 Part 1
- **已會 Markdown：** 可直接從 Part 1 開始，依序學習每個單元
- **有經驗者：** 可直接跳至 Part 3 學習 PDF 中文支援，或 Part 5 學習自動化腳本

## 你將學會

- ✅ Markdown 語法基礎

- ✅ Markdown ↔ Word ↔ HTML 格式互轉

- ✅ LaTeX 數學公式撰寫
- ✅ Mermaid.js 流程圖整合
- ✅ 批次檔 / PowerShell 自動化腳本
